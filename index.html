n8n Auto-Sync Setup
How auto-sync works
Step 1 — In n8n, create a Schedule Trigger set to run at 11:59 PM daily.
Step 2 — Add an HTTP Request node → Vapi API
GET https://api.vapi.ai/call?assistantId=a7fa7099-cd71-4760-94a8-e4060a731d30&limit=1000
Header: Authorization: Bearer YOUR_VAPI_KEY
Step 3 — Add an HTTP Request node → GHL API
GET https://services.leadconnectorhq.com/opportunities/search?location_id=0KudMlNZlDTEfkIM0398
Header: Authorization: Bearer YOUR_GHL_KEY
Step 4 — Add a Code node to aggregate: count calls, hang-ups, durations, stage counts.
Step 5 — Add an HTTP Request node (POST) to your hosted dashboard URL:
POST https://YOUR-DASHBOARD-URL/api/sync
Body: the aggregated JSON object (format shown in Config panel → Data Format tab).
Note — This dashboard also accepts a manual Refresh Now push from n8n via the same endpoint. Keep your GHL + Vapi API keys in n8n credentials only — never paste them into this page.
